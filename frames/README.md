# Frame Overlays

Place your custom frame PNG files in this directory. After adding frames:

1. Run `npm run setup` to copy them to `client/public/frames/`
2. Register each frame in `client/src/App.jsx` → the `FRAMES` array

## Frame Specification

| Property    | Value                              |
|-------------|------------------------------------|
| Format      | PNG with alpha transparency        |
| Dimensions  | **1280 × 720 pixels**              |
| Center area | Fully transparent (shows the photo)|
| Border      | Decorative elements around edges   |
| File naming | `kebab-case.png` (e.g. `gold-party.png`) |

## Included placeholder frames

These are auto-generated by `npm run setup` using Sharp:

- `gold-party.png`   — Gold border
- `neon-glow.png`    — Cyan neon border
- `classic-polaroid.png` — White border (Polaroid style)

## Adding a custom frame

1. Create a 1280×720 PNG in your image editor (Figma, Photoshop, Canva, etc.)
2. Keep the centre of the image **fully transparent** — that's where the photo shows through
3. Add decorations (borders, stickers, text) around the edges
4. Save as PNG and drop it here
5. Run `npm run setup` (copies to `client/public/frames/`)
6. Add an entry to the `FRAMES` array in `client/src/App.jsx`:

```js
{ slug: 'my-frame', label: 'My Frame', thumbnail: '/frames/my-frame.png' }
```
